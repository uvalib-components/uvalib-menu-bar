# \<uvalib-menu-bar>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i uvalib-menu-bar
```

## Usage
```html
<script type="module">
  import 'uvalib-menu-bar/uvalib-menu-bar.js';
</script>

<uvalib-menu-bar></uvalib-menu-bar>
```
